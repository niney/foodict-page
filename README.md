# Foodict

요리 레시피를 공유하고 탐색할 수 있는 웹사이트입니다.

## 소개

Foodict는 다양한 요리 레시피를 한눈에 볼 수 있는 플랫폼입니다. 간단한 가정식부터 특별한 요리까지, 누구나 쉽게 따라할 수 있는 레시피를 제공합니다.

## 기술 스택

- Svelte 5
- Vite 6
- GitHub Pages + GitHub Actions (자동 배포)

## 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 미리보기
npm run preview
```
